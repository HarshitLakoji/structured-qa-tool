Question: What is a Data Structure? Explain the difference between primitive and non-primitive data structures with examples.
Answer: A Data Structure is a method to organize and store data in a computer program to allow efficient access and use. Primitive data structures, like integers, floats, characters, and Booleans, are the most basic structures available in a programming language. They are simple and are not built on other data types. For example, an integer represents whole numbers, a float represents numbers with decimals, a character represents letters or symbols, and a Boolean represents true/false values.


Non-primitive data structures are more complex and are built using primitive data types. They include arrays, linked lists, stacks, queues, trees, graphs, and hash tables. For example, an array can store a collection of primitive data types, while a tree can store data in a hierarchical structure, like a family tree.


In essence, the key difference is that primitive data structures are the simplest and are not composed of other data structures, whereas non-primitive data structures are built from primitive types and can represent more complex relationships.
